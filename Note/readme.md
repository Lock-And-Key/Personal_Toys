## TODO提醒

### why ?

刚开始就是简单地想，有时候，能够在特定的时候有人短信通知我要做的事，我应该就不会忘了吧。于是写了这么一个WebAPP。添加提醒，到时候发送短信提醒我。为此，充值了50元的短信费。

### more

后来，学习vue时，用vue重写了这个WebAPP。也算是入门vue了。
